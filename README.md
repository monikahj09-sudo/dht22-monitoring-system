# dht22-monitoring-system
IoT-based Temperature and Humidity Monitoring System using ESP32 and DHT22 sensor. This project reads real-time environmental data and displays temperature and humidity values on the Arduino IDE Serial Monitor. Designed for smart home, weather monitoring, greenhouse, and IoT applications.
# ESP32 DHT22 Temperature and Humidity Monitoring System

## Overview
This project uses an ESP32 microcontroller with a DHT22 sensor to monitor real-time temperature and humidity values. The data is displayed on the Arduino IDE Serial Monitor.

---

## Components Required
- ESP32 Dev Module
- DHT22 Sensor
- Jumper Wires
- USB Cable
- Arduino IDE

---

## Circuit Connections

| DHT22 Pin | ESP32 Pin |
|-----------|------------|
| VCC       | 3.3V       |
| GND       | GND        |
| DATA      | GPIO 4     |

---

## Features
- Real-time temperature monitoring
- Humidity sensing
- Serial Monitor display
- Low-cost IoT implementation
- Easy ESP32 integration

---

## Software Used
- Arduino IDE
- ESP32 Board Package
- DHT Sensor Library

---

## Output
The Serial Monitor displays:
- Temperature in Celsius
- Humidity Percentage

Example:
Temperature: 34.70 °C
Humidity: 51.00 %
